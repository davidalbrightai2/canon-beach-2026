# Cannon Beach 2026 - Albright Family Trip

Trip planning app for our Cannon Beach adventure, April 1-5, 2026.

## Features
- 📅 5-day overview with best weather and tide predictions
- 🍽️ Meal planning and coordination
- 🎯 Activity suggestions with voting
- 🌊 Real-time tide information
- ☀️ Weather forecasts

## How to Use
Visit the live app at: https://davidalbrightai2.github.io/cannon-beach-2026/

The app connects to a shared Google Sheet for real-time updates across all devices.
